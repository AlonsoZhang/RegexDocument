# RegexDocument

### 使用说明

![regex.png](http://www.zhangwu.tech/regex.png)

1. 复制文本到此处
2. 输入正则表达式
3. 得到所有符合正则的结果
4. 统计符合正则的个数
5. 将结果输出到桌面csv
6. 结果展示
7. 文件路径
8. 文件选择(txt,log)
9. 输入正则表达式
10. 得到第一次匹配的行数
11. 显示结果
12. 行数加500，也可自己修改
13. 删除行数后的内容
14. 输入需要删除的正则表达式
15. 快捷输入帮助
16. 输出匹配的所有行数
17. 删除匹配内容
18. 选择文件1
19. 选择文件2
20. 比对文件1和文件2，将相同的行删除
